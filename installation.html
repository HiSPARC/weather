1. InstallationÂ¶

The Weather DAQ is installed along with the other HiSPARC software when using the HiSPARC installer. The Weather Station DAQ is a LabView program and requires the LabView Runtime Engine to be installed. Furthermore, to communicate with weather stations, the softare requires that the proper drivers are installed, these are often provided with the weather station hardware.
